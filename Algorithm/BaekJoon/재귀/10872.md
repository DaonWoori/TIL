# 팩토리얼

### 문제❓
```
0보다 크거나 같은 정수 N이 주어진다. 이때, N!을 출력하는 프로그램을 작성하시오.
```

### 제한사항⚠️


### 입출력⌨️
* 첫째 줄에 정수 N(0 ≤ N ≤ 12)이 주어진다.
* 첫째 줄에 N!을 출력한다.

#### 입출력 예시
```
# 입력 예시1
10
# 출력 예시1
3628800

# 입력 예시2
0
# 출력 예시2
1
```

### 내 코드🖥️
```
def factorial(n):
  if n <= 1:
    return 1
  return n * factorial(n-1)


N = int(input())

print(factorial(N))
```

# 합

## 문제 설명
n이 주어졌을 때, 1부터 n까지 합을 구하는 프로그램을 작성하시오.

## 제한 조건
* 첫째 줄에 n (1 ≤ n ≤ 10,000)이 주어진다.
* 1부터 n까지 합을 출력한다.

## 예시
#### 입력
```
3
```

#### 출력
```
6
```
 
## 코드
```
n = int(input())
sum = 0

for i in range(1, n+1):
  sum += i

print(sum)
```

## 다른 사람 풀이
```
print(n * (n + 1) // 2)
```

### 리뷰
> for문을 사용하지 않고, 합공식을 활용하는 방법

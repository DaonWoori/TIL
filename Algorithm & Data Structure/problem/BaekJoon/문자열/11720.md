# 숫자의 합

## 문제 설명
N개의 숫자가 공백 없이 쓰여있다. 이 숫자를 모두 합해서 출력하는 프로그램을 작성하시오.

## 제한 조건
* 첫째 줄에 숫자의 개수 N (1 ≤ N ≤ 100)이 주어진다. 둘째 줄에 숫자 N개가 공백없이 주어진다.
* 입력으로 주어진 숫자 N개의 합을 출력한다.

## 예시
#### 입력
```
5
54321
```

#### 출력
```
15
```
 
## 코드
```
import sys
N = int(sys.stdin.readline())
num_str = sys.stdin.readline().strip()
sum = 0

for n in num_str:
  sum += int(n)

print(sum)
```
